Download

The latest version of ReTux can be downloaded via any of the following official sources:

  • Main Release Page (an Atom feed is also available)
  • GameJolt page
  • itch.io page

ReTux has also been packaged by some third parties, though please note that some of these may be out-of-date as we do not control them (please feel free to let us know if you maintain a ReTux package not listed here):

  • AUR: retux-bin
  • GNU Guix: retux
  • Mageia: retux
  • OpenSUSE Build Service: games/retux
  • SlackBuilds: games/retux
